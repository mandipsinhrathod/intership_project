Smart Waste Management

This project introduces an innovative solution for urban waste collection. The Smart Waste Management system integrates AI-powered sorting, real-time tracking via GPS sensors, and a reward system to encourage recycling. It also features an admin panel, emergency waste request feature, and gamified recycling tips for users. The app was designed to serve Jamnagar City, combining IoT sensors and blockchain for transparent waste handling. Built using HTML, CSS, JavaScript, and PHP with MySQL.
