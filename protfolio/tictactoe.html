TicTacToe

This project is a fun and interactive TicTacToe game built with HTML, CSS, and JavaScript. It allows two players to take turns, detects win/tie conditions, and provides an option to restart the game. The interface is responsive, clean, and designed to offer an enjoyable experience on both desktop and mobile devices. It was developed to strengthen JavaScript logic, DOM manipulation, and event handling skills.
