Shoezy

Shoezy is a modern, responsive website designed for a shoe cleaning service. It allows users to browse cleaning packages, schedule appointments, and read customer testimonials. Built using HTML, CSS, JavaScript, and Bootstrap, the platform ensures compatibility across devices. Interactive features such as price cards, modals for booking, and a contact form enhance the user experience. It was created as part of a college web development project and focuses on clean UI and service-driven design.
